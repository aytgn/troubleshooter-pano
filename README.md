# 🛠️ Troubleshooter Kayıt Panosu (Dockerized)

Bu proje; Nginx, Apache (PHP) ve PostgreSQL kullanılarak oluşturulmuş, tamamen Docker Compose üzerinde koşan izole bir web uygulamasıdır.

## 🚀 Nasıl Çalıştırılır?

Sisteminizde Docker ve Docker Compose kurulu olması yeterlidir. Terminalde proje dizinine girin ve şu komutu çalıştırın:

bash docker compose up -d --build

Tarayıcınızdan http://localhost:8080 adresine giderek uygulamayı kullanmaya başlayabilirsiniz.
